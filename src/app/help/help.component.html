Help Information for Users

Welcome

Welcome to our site! Here's a guide to help you navigate through the different sections and make the most out of your experience:

Dashboard

The Dashboard is your main landing page. Here, you can get an overview of all the features available on the site. It provides a starting point to explore the various sections, including games and admin functionalities.

Admin User Page

The Admin User Page is where you can manage content. This includes adding new categories and words for each category. As an admin, you have the control to expand the vocabulary and customize the game experience for users.

Let's Play Page

The Let's Play Page showcases all the games available on the site. Here, you can explore and choose from the list of games. Each game offers a unique experience:

  • Selecting a Game: Click on a game to open a dialog box. From there, you can choose the categories available for that game. This allows you to select the specific category you'd like to play, tailoring the experience to your preference.

User Profile and Admin Access

In the top right corner of the main header, there is an icon of a person. Clicking on this icon will take you to your personal admin page. Here, you can access and manage your admin settings and content contributions.

Need Further Assistance?

Feel free to explore and enjoy the various games and features our site offers. If you have any questions or need further assistance, please don't hesitate to reach out through the help icon at the bottom of the page. Have fun!
